# Veer-Zaara Video App

Ek khoobsurat Flutter app jisme Veer-Zaara theme ka video player interface banaya gaya hai. Yeh app mobile, web aur desktop par chalta hai aur isme ek video player ka mockup aur aage dekhne ke liye kuch video suggestions ki list di gayi hai.

## Features

- **Video Player Layout**: Veer-Zaara theme ka ek sundar video player mockup.
- **Cross-Platform**: Mobile, web, aur desktop par perfectly kaam karta hai.
- **Responsive Design**: Sabhi screen sizes ke hisaab se adjust hota hai.
- **Related Videos**: Niche aur bhi videos dekhne ka option.

## Tech Stack

- Flutter
- Dart
- Material Design 3

## Setup aur Run Kaise Karein

1. Flutter SDK install karein.
2. Repo clone karein aur terminal mein run karein: `flutter pub get`
3. App run karne ke liye: `flutter run`
